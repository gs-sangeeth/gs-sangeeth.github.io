Sky Golf

Custom controller


Gameplay


The brief of this project was to create a custom conytroller using sensors and an arduino, and then using that custom controller develop a game in Unity. My controller had a peizoelectric sensor, potentiometer and touch sensor. I used it to make a mini golf game where the potentiometer knob comtrols the rotation of the golf ball, and peizoelectric sensor was used to get the force input for knocking the ball.
